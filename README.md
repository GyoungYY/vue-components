# vue-components

> 这是一个vue组件库，收集在vue中适用的实用组件

## 开发
```bash
    # 克隆项目
    git clone git@github.com:GyoungYY/vue-components.git

    # 安装依赖
    npm install
    //or # 建议不要用cnpm  安装有各种诡异的bug 可以通过如下操作解决npm速度慢的问题
    npm install --registry=https://registry.npm.taobao.org

    # 本地开发 开启服务
    npm run dev
```
